## EC2란?

Amazon EC2(Elastic Compute Cloud)는 AWS에서 제공하는 **가상 서버** 서비스입니다.

## 인스턴스 생성

### 1. AMI 선택

```bash
# Amazon Linux 2023 또는 Ubuntu 22.04 LTS 권장
# 프리 티어: t2.micro (1 vCPU, 1GB RAM)
```

### 2. 보안 그룹 설정

| 포트 | 프로토콜 | 용도 |
|------|----------|------|
| 22 | SSH | 서버 접속 |
| 80 | HTTP | 웹 트래픽 |
| 443 | HTTPS | 보안 웹 트래픽 |
| 3000 | TCP | Next.js 개발 서버 |

### 3. SSH 접속

```bash
# 키 페어 권한 설정
chmod 400 my-key.pem

# SSH 접속
ssh -i my-key.pem ec2-user@<퍼블릭-IP>
```

## Docker 설치

```bash
# Amazon Linux 2023
sudo yum update -y
sudo yum install -y docker
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker ec2-user

# Docker Compose 설치
sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
```

## 애플리케이션 배포

```bash
# 프로젝트 클론
git clone https://github.com/your-repo/blog.git
cd blog

# Docker Compose로 실행
docker-compose up -d

# 상태 확인
docker-compose ps
```

## 팁

- **Elastic IP**를 할당하면 인스턴스 재시작 시에도 IP가 유지됩니다
- **프리 티어**는 월 750시간 무료 (t2.micro 기준)
- 비용 절약을 위해 사용하지 않을 때는 인스턴스를 **중지**하세요
